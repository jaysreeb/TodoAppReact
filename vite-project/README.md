# React Todo App

A beginner-friendly Todo App built with React.  
You can add new tasks and mark them as completed using checkboxes.

## Features
- Add new tasks using the input box
- Mark tasks as completed with a checkbox
- Tasks show a line-through when completed

## Technologies Used
- React (useState hook)
- HTML & CSS (basic)
